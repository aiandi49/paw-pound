# 🏆 Paw Pound — Hot or Not · For Pets

> **Tucson's community pet rating app**  
> Built by Lamar Myers · VTREI LLC · Tucson, Arizona

---

## What It Does

Paw Pound is the fun layer of PetHub. Post your pet, get rated HOT or NOT by the Tucson community, climb the leaderboard, and build daily voting streaks.

It's Hot or Not — but make it pets. Clean, fast, addictive, and built with real dog photos from a live photo API.

---

## Features

| Feature | Description |
|---|---|
| 🔥 **HOT or NOT Voting** | Swipe-style voting on pet photos submitted by the community |
| 🏆 **Live Leaderboard** | Real-time Hot Board ranking Tucson's top-rated pets |
| 📸 **Real Dog Photos** | Live photos fetched from dog.ceo API — 20,000+ real breed photos |
| 🎉 **Confetti Moments** | Confetti animation fires on milestone votes and big matches |
| ⚡ **Live Vote Feed** | Watch votes come in across the community in real time |
| 🔁 **Daily Streaks** | Return every day to maintain your voting streak |
| 🌙 **Dark Editorial Design** | Instrument Serif + DM Mono — sophisticated, not childish |

## How It Works

1. **Browse the deck** — swipe through pet photos
2. **Vote HOT or NOT** — your votes affect the leaderboard in real time
3. **Watch the Hot Board** — see which pets are climbing the ranks
4. **Submit your pet** — post your own pet to get rated
5. **Build your streak** — come back daily to keep your streak alive

---

## Tech Stack

- **Frontend:** Vanilla HTML, CSS, JavaScript — single-file app
- **Photos:** [dog.ceo API](https://dog.ceo/dog-api/) — free, no key required
- **Design:** Instrument Serif · DM Mono · Dark `#0C0C12` palette
- **Hosting:** GitHub Pages / Vercel
